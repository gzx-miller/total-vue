# Total Vue

一个中文 Vue3 案例学习库。项目使用 `Vue 3 + TypeScript + Vite + Vue Router + Pinia`，通过多路由章节展示 Vue3 的重要知识点。

## 启动

```bash
pnpm install
pnpm dev
```

## 验证

```bash
pnpm type-check
pnpm test
pnpm build
```

## 学习顺序

左侧导航按 `K_1` 到 `K_21` 排列。每个章节都是独立页面，右侧包含案例演示、关键代码、原理、处理流程、注意事项和解决的问题。

核心章节：

- `K_1` 应用创建、入口挂载、SFC 基础结构
- `K_2` 模板语法、插值、指令、事件绑定
- `K_3` 响应式基础
- `K_4` 计算与监听
- `K_5` 条件与列表渲染
- `K_6` 表单处理
- `K_7` 组件基础
- `K_8` 插槽
- `K_9` 组件通信
- `K_10` 生命周期与 DOM 引用
- `K_11` 组合式函数
- `K_12` 路由
- `K_13` Pinia 状态管理
- `K_14` 异步请求与加载状态
- `K_15` 动态组件与缓存
- `K_16` 过渡动画
- `K_17` 自定义指令
- `K_18` Teleport
- `K_19` Suspense 与异步组件
- `K_20` 性能与工程实践
- `K_21` 测试与可维护性

## 约定

- 页面展示使用中文，变量名和文件名使用英文。
- 每个知识点只解决一个核心问题，避免重复讲解。
- 新增案例时保持 `K_序号` 命名，便于搜索和按顺序学习。
- 提交前至少运行 `pnpm type-check` 和 `pnpm build`。
