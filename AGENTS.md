# AGENTS.md

## 项目目标

这个仓库是“小松鼠举栗子”中文前端知识案例库。当前重点是 Vue3：通过独立真实案例学习 Vue3 重要知识点。不要把它改成营销页、模板站或杂乱示例集合。

## 技术栈

- Vue 3
- TypeScript
- Vite
- Vue Router
- Pinia
- Vitest

## 编码规则

- UI 文案和章节说明使用中文。
- 文件名、变量名、类型名使用英文。
- 路由必须保留知识类别层级；Vue3 分类使用 `/vue/...`。
- 视觉风格应贴合秋日森林、红橙枫叶、暖光、小松鼠抱栗子的参考图气质。
- 每个 `K_` 案例只覆盖一个主要知识点，避免重复讲同一概念。
- 左侧导航使用简洁短标题，正文标题和说明可以更详实。
- 案例演示优先使用真实小业务场景，例如购物车、报名表、课程列表、弹窗、搜索。
- 关键代码应简短可读，完整代码保留在对应组件中。
- 组件内部逻辑优先使用 `<script setup lang="ts">`。
- 可复用状态逻辑放到 `src/composables`，跨页面共享状态放到 `src/stores`。

## 新增案例流程

1. 在 `src/demos` 新增独立 demo 组件。
2. 在 `src/data/lessons.ts` 注册新的 `K_序号`、短标题、分类路由、说明和代码片段。
3. 保证左侧导航可以完整展示短标题。
4. 高风险逻辑补测试，至少覆盖 store、composable 或组件交互中的一种。
