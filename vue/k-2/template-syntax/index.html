模板语法、插值、指令、事件绑定

用订单搜索展示插值、v-model、v-if、v-for、属性绑定和事件绑定如何协作。

关键代码

<script setup lang="ts">
import { computed, ref } from 'vue'

const keyword = ref('拿铁')
const orders = ['冰拿铁', '燕麦拿铁', '美式', '手冲']
const filteredOrders = computed(() => orders.filter((order) => order.includes(keyword.value)))

function clearKeyword() {
  keyword.value = ''
}
</script>

<template>
  <div class="demo-card">
    <h3>{{ keyword || '全部' }} 订单搜索</h3>
    <input v-model="keyword" placeholder="输入饮品名" />
    <button :disabled="!keyword" @click="clearKeyword">清空</button>
    <p v-if="filteredOrders.length">找到 {{ filteredOrders.length }} 个订单。</p>
    <p v-else>暂无匹配订单。</p>
    <ul>
      <li v-for="order in filteredOrders" :key="order">{{ order }}</li>
    </ul>
  </div>
</template>

原理

Vue 模板是一层声明式视图描述：状态是什么，页面就应该长什么样。插值负责显示文本，指令负责常见 DOM 行为，事件绑定负责把用户动作交回组件逻辑。

处理流程

  1. 输入框通过 v-model 把关键词同步到响应式状态。
  2. computed 根据关键词过滤订单列表，避免在模板中写复杂表达式。
  3. 模板根据过滤结果使用 v-if/v-else 显示反馈，并通过 v-for 渲染列表。

注意事项

  • 模板里可以写表达式，但复杂判断应移到 computed 或函数中，让模板保持可读。
  • v-for 的 key 要稳定且能代表数据身份，不要使用随机数。
  • 事件处理函数只处理本次交互，不要在模板中堆叠过多内联逻辑。
