工具提示：Tooltip 方向与触发

用课程信息提示展示 ElTooltip 的方向、触发方式和富内容插槽。

关键代码

<script setup lang="ts">
import { ElTooltip, ElButton } from 'element-plus'

const tooltipCode = `// ElTooltip 关键属性
<ElTooltip
  content="提示内容"     // 文本内容
  placement="top"        // 方向：top/bottom/left/right
  trigger="hover"        // 触发：hover/click
  :hide-after="2000"     // 自动隐藏延迟
>
  <ElButton>触发元素</ElButton>
</ElTooltip>

// 富内容使用 #content 插槽
<ElTooltip placement="top">
  <template #content>
    <div>多行<br/>提示内容</div>
  </template>
  <ElButton>富内容提示</ElButton>
</ElTooltip>`
</script>

原理

ElTooltip 在鼠标悬停或点击时显示提示信息。placement 控制弹出方向，trigger 控制触发方式（hover/click）。content 属性用于简单文本，#content 插槽支持富内容（多行、链接等）。

处理流程

  1. 用 placement 属性设置提示方向（top/bottom/left/right）。
  2. trigger="click" 改为点击触发，适合需要用户主动查看的场景。
  3. #content 插槽放置多行或结构化内容。

注意事项

  • Tooltip 内容应简洁，复杂信息建议用 Popover。
  • hide-after 控制自动隐藏延迟，设为 0 不自动隐藏。
  • Tooltip 不会阻塞页面交互，适合辅助说明。
