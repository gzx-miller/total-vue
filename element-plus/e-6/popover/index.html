气泡卡片：悬浮触发与嵌套内容

用快捷操作和课程列表展示 ElPopover 的触发方式和嵌套内容。

<script setup lang="ts">
import { ref } from 'vue'
import { ElPopover, ElButton, ElInput } from 'element-plus'

const searchKeyword = ref('')

const courseList = [
  'Vue3 组合式 API 入门',
  'TypeScript 类型编程',
  'Element Plus 组件精讲',
  'Pinia 状态管理',
  'Vite 构建优化',
]
</script>

<template>
  <div class="demo-card">
    <h3>气泡卡片 Popover</h3>

    <h4>基础用法</h4>
    <div class="button-row">
      <ElPopover
        placement="top"
        title="标题"
        :width="200"
        trigger="hover"
      >
        <template #reference>
          <ElButton>悬浮上方</ElButton>
        </template>
        <p>这是一段内容，鼠标移出会自动关闭。</p>
      </ElPopover>

      <ElPopover
        placement="bottom"
        title="快捷操作"
        :width="160"
        trigger="click"
      >
        <template #reference>
          <ElButton type="primary">点击下方</ElButton>
        </template>
        <div class="action-list">
          <ElButton size="small" link>新建课程</ElButton>
          <ElButton size="small" link>编辑资料</ElButton>
          <ElButton size="small" link type="danger">退出登录</ElButton>
        </div>
      </ElPopover>

      <ElPopover
        placement="right"
        title="功能说明"
        :width="220"
        trigger="focus"
      >
        <template #reference>
          <ElInput
            v-model="searchKeyword"
            placeholder="聚焦此处"
            style="width: 140px"
          />
        </template>
        <p>输入关键词后，系统会自动从课程库中搜索相关内容。</p>
      </ElPopover>
    </div>

    <h4>嵌套内容</h4>
    <ElPopover
      placement="bottom-start"
      :width="240"
      trigger="click"

        <ElButton>课程列表</ElButton>
      </template>

        <div class="course-list">

        </div>
      </template>
    </ElPopover>
  </div>
</template>

原理

ElPopover 的触发方式由 trigger 属性控制（hover/click/focus/manual）。reference slot 放置触发源，默认 slot 放置气泡内容。

处理流程

  1. hover 触发适合工具提示和快捷操作。
  2. click 触发适合需要展开详细操作的场景。
  3. 气泡内容通过默认 slot 传入，支持复杂布局和交互。

注意事项

  • 气泡卡片会被窗口边缘截断，Element Plus 会自动调整位置。
  • 嵌套内容时应避免气泡内再嵌套气泡。
  • trigger=manual 需要手动控制显示状态。
