级联选择：Cascader 多级联动

用课程分类选择展示 ElCascader 的多级联动和路径显示。

<ElCascader

  :options="options"          // 级联数据源
  :props="{ expandTrigger: 'hover' }"  // 悬浮展开

原理

ElCascader 通过 options 属性接收树形数据，每级包含 value、label 和 children。v-model 绑定值为各级 value 组成的路径数组。expandTrigger 控制子级展开方式（click/hover）。

处理流程

  1. 定义树形 options 数据，每级有 value、label 和 children。
  2. v-model 绑定选中路径数组，如 ["frontend", "vue", "vue3-basics"]。
  3. 选择完成后显示完整路径，可通过 props.emitPath=false 只获取最后一级值。

注意事项

  • 数据量大时考虑异步加载（props.lazy + props.lazyLoad）。
  • emitPath=false 时 v-model 只绑定最后一级的 value。
  • 可配合 filterable 属性支持搜索过滤。
